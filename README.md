# COMP3004 Group Project
## Travel App

An app that compiles all of your travel info and trip information
to make the process of travelling to other countries easy.

 * Travel Itinerary
 * Reminders
 * Packing List:
    * Templates
    (i.e. Camping trip, Beach trip, ski trip, city visit)
  * Day planner
    * Currency Tracker
    * Budget Tracker
